# 多链支持

UniPass Wallet 可以支持所有 EVM 兼容链，并通过 [**EIP-2470**](https://eips.ethereum.org/EIPS/eip-2470) 保持多链同一钱包地址。目前支持的链如下：

### 主网

- Ethereum
- BNB Chain
- Polygon
- Rangers Protocol
- Arbitrum
- Avalanche (C chain)
- KCC
- PlatON
- OKC

### 测试网

- Goerli (Ethereum testnet)
- BSC testnet
- Mumbai (Polygon testnet)
- Robin (Rangers testnet)
- Arbitrum Goerli
- Avalanche Fuji testnet
- KCC testnet
- OKC testnet
- PlatON testnet
- Scroll

## 未来将支持的链
